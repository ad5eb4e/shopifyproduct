# Shopify 产品管理系统 — 运营手册

## 1. 系统简介

本系统做两件事：

```
① 同步：Shopify 店铺 ──→ 自动拉取产品数据 ──→ 飞书多维表

② 回写：飞书多维表（合规状态="已改写"）──→ 自动回写改写内容 ──→ Shopify 店铺
```

合规检查和改写由你在外部完成（使用合规改写工具）。本系统只负责**拉数据**和**回写**。

**你需要做的事：**
1. 在管理后台配置店铺信息（一次性）
2. 在飞书多维表中查看产品数据
3. 使用合规工具检查和改写产品（外部完成）
4. 改写完成后系统自动回写，无需操作

详细功能说明见 [产品同步模块说明](apps/sync/GUIDE.md)。

## 2. 管理后台操作

### 2.1 登录

1. 浏览器打开管理后台地址（部署时提供）
2. 输入用户名和密码
3. 30 分钟无操作自动退出

### 2.2 添加店铺

在管理后台点击「+新增」，填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 店铺名称 | 自定义名称，飞书中区分不同店铺 | US-01 |
| 店铺标识 | Shopify 后台域名前缀 | us-store-01 |
| 前端域名 | 客户访问的网站地址 | www.us-store.com |
| 认证方式 | Token 或 OAuth（见第 3 节） | Token |
| Access Token | Token 认证时填写 | shpat_xxxx |
| Client ID / Secret | OAuth 认证时填写 | |
| 代理IP | 该店铺专用代理 | socks5://1.2.3.4:1080 |
| 飞书 Table ID | 留空用全局默认表格，填写则用独立表格 | |

### 2.3 常用操作

| 操作 | 说明 |
|------|------|
| 「测试连接」 | 验证凭证和代理是否正常 |
| 「立即同步」 | 手动触发一次产品数据同步 |
| 「立即回写」 | 手动触发一次回写 |
| 「查看日志」 | 查看操作记录，可按功能筛选（同步 / 回写） |

### 2.4 查看系统健康状态

管理后台首页可以看到：
- 系统运行状态（正常 / 异常）
- 每个店铺最后一次成功同步的时间
- 代理连接状态

## 3. 如何获取 Shopify API 凭证

### 方式一：Access Token（2026.1.1 前已创建 Custom App）

1. Shopify 后台 → Settings > Apps > Develop apps > Create an app
2. Configure Admin API scopes：`read_products`、`write_products`、`read_product_listings`
3. Save → Install app → 复制 Access Token（只显示一次）

### 方式二：OAuth（2026.1.1 后新店铺）

1. [Shopify Partners Dashboard](https://partners.shopify.com) → Create app
2. 配置 scopes → 获取 Client ID + Client Secret
3. 安装到店铺 → 管理后台选 OAuth → 填入 Client ID 和 Secret
4. 系统自动获取和续期 Token

## 4. 飞书多维表说明

### 4.1 表格结构

默认所有店铺共用一张多维表，通过"店铺名称"区分。

### 4.2 字段说明

**基础信息（系统自动同步）：**

| 字段 | 说明 |
|------|------|
| 店铺名称 | 来自哪个店铺 |
| Shopify产品ID | 产品唯一编号 |
| 产品状态 | active / draft / archived / 已删除 |
| 价格 | 所有规格的价格，每行：`规格名 | 价格` |
| SKU | 所有规格的 SKU，每行：`规格名 | SKU` |
| 前端链接 | 产品在网站上的访问地址 |
| 产品图片 | 所有主图 URL，换行分隔 |
| Variant图片 | 每行：`规格名 | 图片URL` |

**可扫描字段（原始值 + 改写值成对出现）：**

| 原始字段 | 改写字段 | 说明 |
|---------|---------|------|
| 产品标题 | 改写标题 | 产品名称 |
| 原始描述HTML | 改写描述 | HTML 格式 |
| 原始描述纯文本 | — | 去格式版，便于阅读 |
| 产品类型 | 改写产品类型 | |
| 产品标签 | 改写标签 | 逗号分隔 |
| SEO标题 | 改写SEO标题 | |
| SEO描述 | 改写SEO描述 | |
| Handle | 改写Handle | URL 路径 |
| 图片Alt文本 | 改写图片Alt | 换行分隔 |
| 规格名称 | 改写规格名称 | 换行分隔 |
| 所属集合 | — | 只读展示 |

**状态字段：**

| 字段 | 说明 |
|------|------|
| 合规状态 | **待检查** — 等待你检查 / **合规** — 无需改写 / **已改写** — 改写完成，系统将自动回写 |
| 回写状态 | **未回写** — 初始 / **已回写** — 已更新到 Shopify / **回写失败** — 失败 |
| 最后更新时间 | 最后一次更新时间 |

### 4.3 工作流程

```
[系统] 同步产品 → 飞书中合规状态 = "待检查"
                        ↓
[你] 使用合规工具检查，合规 → 改为"合规"
                  需改写 → 填写改写字段 → 改为"已改写"
                        ↓
[系统] 检测到"已改写" → 自动回写 Shopify → 回写状态 = "已回写"
                                         → 合规状态 = "合规"
```

## 5. 常见问题

### 回写失败怎么办？
查看日志，常见原因：凭证过期、代理异常、Shopify 限速。

### 如何重新回写？
将"回写状态"改回"未回写"，系统下次检查时自动重新回写。

### 产品在 Shopify 被删了？
系统标记"已删除"，保留记录但不再回写。

### 在 Shopify 后台手动改了产品？
下次同步自动检测到变化，更新飞书并重置为"待检查"。

### 改写字段格式
- **改写描述**：HTML 格式
- **改写标签**：逗号分隔
- **图片Alt、规格名称**：换行分隔
- **其他**：纯文本
- **不需要改的留空**

---

## 6. 注意事项

1. **代理IP**：每个店铺独立代理，互不关联
2. **凭证安全**：Token / Secret 请勿分享给无关人员
3. **首次同步**：新店铺全量拉取，后续增量
4. **回写时机**：检测到"已改写"后最多 1 小时内自动回写
